== Description ==
Displays categories and subcategories as a mindmap in a widget


== Installation ==

1. Upload `mindcat` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress admin
3. Create a new page and insert the short code `[eelv_news_form]`
4. You can edit defaults settings in Settings > Event post

== Frequently asked questions ==

= Can I select a primary category as mindmap root ?  =
Yes, you can choose any category, all sub-categories will be children of your mindmap
